# 🚲 Bike Crash Counter

A simple, fun, and personal web application to track my motorcycle adventures and log every "oops" moment on the road. This project was built to practice **JavaScript fundamentals**, **DOM manipulation**, and **CSS styling**.

## 🚀 Features
- **Live Counter:** Increments every time I survive a fall.
- **Save Log:** Keeps a history of previous crashes in a dedicated list.
- **Responsive Design:** A cool background image that fits any screen size.
- **Persistence:** Designed to be the first of many personalized utility apps.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure.
- **CSS3:** Custom styling, centering, and responsive backgrounds.
- **JavaScript:** Logic for counting, saving, and UI updates.
